AutoLayoutTableViewSample
=========================

Auto Layoutで作られたセルから自動的に高さを計算する最小限のサンプルです。

- iOS 6以上のiPhoneおよびiPadに対応
- iPhone 6/6 Plusにも対応
- `NSFoundationVersionNumber_iOS_7_1` で条件分岐している箇所がiOS 7.1以下で必要な処理です
